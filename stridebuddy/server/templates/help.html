StrideBuddy Help

Getting started

First, create an account on the signup page, then sign in from the desktop app.

  • Create a screen name
  • Forgot password

Sign in options

Save password stores your password in the system keychain. Auto‑login signs you in automatically next launch if a saved password exists.

Buddy List

Double‑click a buddy to open a message window. Use Sign Off to return to the sign‑on screen.

Messaging

Use the toolbar to format your message (B / I / U, color, link, emoji). Send with Enter or Ctrl+Enter.

Troubleshooting

  • If sign‑in fails, ensure the server is running at http://127.0.0.1:5000.
  • Can’t open links? Make sure your browser is set as default.
