StrideBuddy Help

Getting started

Create a screen name, start the desktop app, and sign in.

  • Create a screen name
  • Forgot password

Sign in options

Save password stores your password in the system keychain. Auto‑login signs you in automatically next launch if a saved password exists. You can manage these in Setup → Account.

Buddy List

  • Double‑click a buddy to open a chat.
  • Right‑click a buddy to Mute/Block or remove; right‑click a group to Rename.
  • Status shows (online), (away), or (offline). Flags [muted]/[blocked] are preserved.

Messaging

  • Use the toolbar (B/I/U, color, link, emoji). Links are clickable on the recipient.
  • Typing indicators display in the chat status bar.
  • Send with Enter or Ctrl+Enter.

Settings

  • Notifications: sounds and desktop toasts.
  • Chat: default styles and Enable transcript logging.
  • Appearance: theme (Light/Retro/Auto), timestamp (12h/24h), compact spacing.
  • Data: open settings folder, export/import settings, clear saved passwords.

Where things are

  • Logs: %APPDATA%\StrideBuddy\logs (enable in Setup → Chat)
  • Settings: %APPDATA%\StrideBuddy\settings.json

Troubleshooting

  • Sign‑in fails: ensure the server is running at http://127.0.0.1:5000.
  • Unauthorized (401): sign out and sign in again.
  • No sounds: enable in Setup → Notifications; check system volume.
  • Help/Signup 404: restart server and hard‑refresh (Ctrl+F5).
